# Sistema-de-Gest-o-de-Alunos
Sistema em Python para análise de engajamento e notas de alunos em infoprodutos.
# Analisador de Desempenho e Engajamento de Alunos

Este é um projeto desenvolvido em Python focado em resolver um desafio real do ecossistema da **Creator Economy**: 
a retenção e análise de sucesso de alunos em plataformas de infoprodutos, inspirado nas soluções de dados.

## Objetivo do Projeto
O script automatiza a análise de dados de estudantes de um curso digital, avaliando suas notas finais e o 
percentual de progresso (aulas assistidas). Com isso, o sistema gera insights automáticos para o Produtor 
de Conteúdo identificar clientes ideais para ações de engajamento ou suporte contra a evasão de alunos.

## Conceitos Técnicos Aplicados
- **Estruturas de Dados:** Uso de listas e dicionários para simular cargas de dados de APIs estruturadas.
- **Lógica Condicional Avançada:** Classificação de padrões de comportamento com `if/elif/else`.
- **Loops e Iterações:** Processamento dinâmico de grandes conjuntos de dados utilizando `for`.
- **Formatação de Strings e Métricas:** Exibição limpa de dados e cálculos de médias aritméticas precisas.
